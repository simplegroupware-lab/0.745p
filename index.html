Update from PHP 5.3.29 to PHP 7.4

  • 1. make a full copy of your sgs directory
  • 2. download this update-script: update_v0745_to_php74.php
  • 3. copy the update-script into directory sgs/bin
  • 4. login as admin and clean the caches
  • 5. run the script sgs/bin/update_v0745_to_php74.php
  • 6. unpack the archive into directory sgs/simple_cache/update_to_php74
  • 7. click Continue
  • 8. switch your php environment to php 7.4 and click Continue again
  • 9. if anything goes wrong, simply restore your backuped sgs directory
  • Support on Google Groups Simple Groupware

Fresh installations

  • download and unpack in the directory where you want to install it
  • call http://your-address/install.php in your browser and follow the instructions
  • needs testing (tested on PHP 7.4.16, MySQL 5.73, Apache)
  • Module PMWIKI/CMS/Intranet is not working, please disable it
